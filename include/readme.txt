Esmska
~~~~~~

A program for sending SMS over the Internet.

 * Homepage: http://esmska.googlecode.com/

 * Requirements: Java 6 (http://www.java.com/)

 * Execution: The program can be run by executing 'esmska.sh' (in Linux, Mac OS, etc.) or 'esmska.exe' (in Windows). It can also be run from the program directory by command 'java -jar esmska.jar' (any operating system). You can get an advanced help by using option '--help'.
